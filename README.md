# Outfit App

A minimal Next.js starter with TypeScript and Tailwind CSS.

## Getting Started

1. Install dependencies:

```bash
pnpm install
```

2. Run the development server:

```bash
pnpm dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy to Vercel

Click the button below to deploy this app to Vercel:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/jakenuts/outfit-app)

Or manually:

1. Push your code to GitHub
2. Import your repository to [Vercel](https://vercel.com)
3. Vercel will auto-detect Next.js and deploy

## What's Included

- **Next.js 15** - React framework with App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first CSS
- **ESLint** - Code linting
